# Terraform Infrastructure

This Terraform configuration provisions the infrastructure required for the TalentIQ AI Candidate Ranking System.

## Resources

- AWS Provider
- EC2 Instance
- Security Group
- SSH Access
- Streamlit Port (8501)

- variables.tf
- terraform.tfvars
- main.tf
- outputs.tf
